# 强化学习值迭代算法 Web 可视化

基于 Flask 和 HTML5 Canvas 的强化学习值迭代算法可视化工具。

## 功能特性

- 🎯 交互式网格世界环境
- 📊 实时显示状态值
- 🎮 策略可视化（箭头显示最优动作）
- 🚀 自动播放功能
- 📈 轨迹可视化

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行应用

```bash
cd web
python app.py
```

然后在浏览器中打开：http://localhost:5000

## 使用说明

1. **初始化环境**：点击"初始化环境"按钮创建网格世界
2. **运行值迭代**：点击"运行值迭代"按钮执行值迭代算法
3. **重置环境**：点击"重置环境"按钮将智能体重置到起始位置
4. **执行一步**：点击"执行一步"按钮让智能体按照最优策略移动一步
5. **自动播放**：点击"自动播放"按钮自动执行步骤直到到达目标

## 界面说明

- **蓝色方块 (T)**：目标状态
- **橙色方块 (X)**：禁止状态
- **绿色方块 (S)**：起始状态
- **红色星号 (★)**：智能体当前位置
- **青色箭头**：最优策略方向
- **紫色线条**：移动轨迹
- **数字**：每个状态的状态值
